# 06_Dev_Plan｜开发计划与版本规划

## 1. 开发原则

1. 每个阶段只做一个目标。
2. 每个阶段完成后必须能运行。
3. 不一次性生成全部功能。
4. 优先完成本地数据闭环。
5. 图谱先做基础版，不追求复杂动效。
6. 部署最后做。
7. 数据导入导出必须尽早完成，避免后续测试丢数据。

## 3. V0.1 项目初始化

### 目标

创建可运行的 Vite + React + TypeScript 项目。

### 任务

- 创建 Vite 项目。
- 安装基础依赖。
- 配置 Tailwind CSS。
- 创建项目目录。
- 创建底部导航。
- 创建四个页面占位。

### 验收

- `npm run dev` 可以运行。
- 页面有图谱、人物、事件、设置四个入口。
- 无报错。

- 实现 initApp。

- 刷新页面不会重复创建。

- 删除人物后关系同步删除。
