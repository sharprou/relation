# 07_Test_Plan｜测试计划与验收标准

## 1. 测试目标

确保关系图谱 PWA 在手机和电脑上能够稳定运行，核心数据可以正确保存、展示、导入和导出。

## 2. 测试环境

### 2.1 桌面端

- Windows 10 / 11
- Chrome 最新版
- Edge 最新版

### 2.2 移动端

- iPhone Safari
- Android Chrome
- iPhone 添加到主屏幕后打开
- Android 添加到主屏幕后打开
