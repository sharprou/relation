⌂添加到主屏幕

在 iPhone / Safari 上

  1. 点击底部“分享”按钮
  2. 选择“添加到主屏幕”
  3. 回到桌面即可像 App 一样打开

在 Android / Chrome 上

  1. 点击右上角菜单按钮
  2. 选择“安装应用”或“添加到主屏幕”
  3. 回到桌面即可快速打开
